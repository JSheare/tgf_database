tgfserver.get_all_configs:
	A function that returns all config info for the specified instrument.

	Parameters
		name TEXT
			The name of the instrument to get the config info for.

	Returns
		TABLE
			afte_date TEXT
				The date after which the config info is valid.
			scint_name TEXT
				The name of the scintillator.
			erc TEXT
				The scintillator's eRC value.
			format_name TEXT
				The name of the list mode date file format used by the scintillator.
			long_event_search BOOL
				A flag indicating whether the scintillator should be considered in the long event search.

tgfserver.get_all_deployments:
	A function that returns all deployments for the specified instrument.
	
	Parameters
		name TEXT
			The name of the instrument to get all deployments for.

	Returns
		TABLE
			start_date TIMESTAMPTZ
				The date at which the deployment begins (inclusive).
			end_date TIMESTAMPTZ
				The date at which the deployment ends (inclusive).
			location TEXT
				The location of the deployment.
			tz_identifier TEXT
				The IANA timezone of the deployment.
			weather_station TEXT
				The callsign of the deployment's nearest weather station.
			sounding_station TEXT
				The identifier of the deployment's nearest sounding station.
			latitude DOUBLE PRECISION
				The approximate latitude coordinate of the deployment (N).
			longitude DOUBLE PRECISION
				The approximate longitude coordinate of the deployment (E, 0-360).
			altitude DOUBLE PRECISION
				The approximate altitude of the deployment (km).
			notes TEXT
				Notes about the deployment.

tgfserver.get_check_ins:
	A function that returns check in information for all instruments.

	Returns
		TABLE
			instrument_name TEXT
				The name of the instrument.
			last_check_in TIMESTAMPTZ
				The timestamp of the last check in.
			storage_fraction DOUBLE PRECISION
				The fraction of storage space used on the instrument.
			has_gps BOOLEAN
				Whether the instrument is receiving gps signal.

tgfserver.get_missing_weather_dates:
	A function that returns the missing weather dates for the given instrument on the given range.

	Parameters
		instrument TEXT
			The name of the instrument that missing weather dates will be listed for.
		start_date TIMESTAMPTZ
			The beginning of the range on which missing dates will be checked for.
		end_date TIMESTAMPTZ
			The end of the range on which missing dates will be checked for.

	Returns
		TABLE
			missing_date TIMESTAMPTZ
				The missing weather info date.

tgfserver.get_scintillators:

	Returns

			scint_priority INTEGER

tgfserver.get_weather:
	A function that returns the weather information for the given instrument on the given date.

	Parameters:
		instrument TEXT
			The name of the instrument to return weather info for.
		target_date TIMESTAMPTZ
			The date that weather info will be retrieved for.

	Returns
		TABLE
			measurement_time TIMESTAMPTZ
				The time that the weather measurement was taken at.
			condition TEXT
				The weather condition at the time of the measurement.
